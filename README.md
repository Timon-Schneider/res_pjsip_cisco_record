# res_pjsip_cisco_record
Asterisk PJSIP module for FreePBX 17 that enables the Record button on Cisco CP-8811, CP-8841, CP-8851, CP-8861 and other CP-8xxx IP phones. Intercepts proprietary x-cisco-remotecc REFER requests, toggles Asterisk MixMonitor to record the call to disk, and cleanly updates the phone's display with the native recording status.
